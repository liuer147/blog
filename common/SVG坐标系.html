    # 视口坐标系

    SVG 使用的坐标系和 canvas 差不多。都是以左上角为`(0, 0)`坐标原点，坐标以像素为单位，x 轴正方向是向右，y 轴正方向是向下。

    <svg>元素默认宽为300px，高为150px。 默认情况下，网格中的一单元相当于 svg 元素中的一像素，而 svg 的一个像素对应输出设备上的一个像素。（svg 还支持其他单位）

    # 用户坐标系

    这是默认的视口坐标系。在 SVG 中还有一个 用户坐标系的概念。不过它默认和视口坐标系相同。 需要知道，用户在 svg 中绘制的图形宽、高是相对于这个用户坐标系的。最终会按照比例缩放、扩大到视口坐标系中。 用户坐标系是通过<svg>元素的viewbox属性指定。 viewbox="startX startY width height"：指定画布中可以显示的区域：从(startX, startY)点开始，100宽 * 100高 的区域。 grid.png（1 个格子 10px，总体 100 * 100）

    # 示例二
      <svg width="300" height="300"  viewBox="50 50 100 100">
    <circle cx="50" cy="50" r="50"></circle>
  </svg>

    最终只展示了右下四分之一个圆。怎么解释？ 同样的，在画布上绘制了坐标是以(50, 50)点为圆心，50px为半径的圆。不过由于指定了画布区域(50, 50) -(50 + 100, 50 + 100)可见。而圆只有右下四分之一个圆在可见部分，则截取该部分，并映射到画布区域。

    # 示例三

    当viewbox指定的可见区域 与 画布的宽高不成比例该怎么办？

    1. 保留可见区域viewbox的宽高比，拉伸/缩小保证可见区域全部在画布中。
    2. 调整可见区域viewbox位置处于画布的垂直和水平居中
