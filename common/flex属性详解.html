    flex属性是flex-basis、flex-grow、flex-shrink三个属性的缩写形式。当父元素是flex布局时display:flex;可以设置生效。flex: <flex-grow> <flex-shrink> <flex-basis>;

    • flex-basis

    **控制元素在主轴方向上初始大小。**可以是以下值：

    1. 长度值18px。（如果是数值0，则初始大小就是0px）
    2. 百分比。（相对于容器的主轴而言）
    3. 关键字auto。（元素本身的width或者height）
    • flex-grow

    在通过**flex-basis**为每一项设置了首选大小之后，如果还有剩余空间，则该系数表示如何处理。其值是一个数值，表示剩余空间的比值。默认值是**0**。

    • flex-shrink

    与**flex-grow**相反，表示空间不够，该元素如何收缩。默认值为 1。

    那么，确定最终尺寸确认的步骤为：

    1. 检查flex-basis，确定初始主尺寸。
    2. 确定实际的主尺寸。如果按照初始尺寸把各项排布好之后，容器还有剩余空间，则根据flex-grow系数决定；如果空间不够，则根据flex-shrink系数决定各项如何收缩。

    如果flex-basis的值为0，那么则该项初始大小为0px，即使它本来存在width 或 height。直接进行第二步的flex-grow与flex-shrink

    那么，是怎么根据flex-grow进行计算，来分配剩余空间的呢？ 其实就是将剩余空间根据各项的flex-group按照比例分配。

    如何根据flex-shrink进行收缩？

    1. 确定收缩值。将超出的值乘以该项的flex-shrink与flex-basis的乘积除以各项flex-shrink与flex-basis的乘积和。
    .container {
  dispplay: flex;
  width: 1000px
}

.div1 {
  flex: 1 1 800px;
}

.div2 {
  flex: 1 1 500px;
}

/* 根据计算公式  */
/* overValue * ((flexShrink1 * flexBasis1) / (flexShrink1 * flexBasis1 + flexShrink2 * flexBasis2)) */

=> 300 * ((1 * 800) / (1 * 800 + 1 * 500)) = 184.6
