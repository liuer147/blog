    # g

    参考：https://developer.mozilla.org/zh-CN/docs/Web/SVG/Element/g (opens new window)

    元素g是用来组合对象的容器。 添加到g元素上的变换会应用到其所有的子元素上。 添加到g元素的属性会被其所有的子元素继承。 此外，g元素也可以用来定义复杂的对象，之后可以通过use (opens new window)元素来引用它们。 利用这个助手，你可以把属性赋给一整个元素集合。实际上，这是它唯一的目的。

    <svg width="100%" height="100%" viewBox="0 0 95 50"
     xmlns="http://www.w3.org/2000/svg">
  <g stroke="green" fill="white" stroke-width="5">
    <circle cx="25" cy="25" r="15" />
    <circle cx="40" cy="25" r="15" />
    <circle cx="55" cy="25" r="15" />
    <circle cx="70" cy="25" r="15" />
  </g>
</svg>

    # <defs>

    参考：https://developer.mozilla.org/zh-CN/docs/Web/SVG/Element/defs (opens new window)

    SVG 允许我们定义以后需要重复使用的图形元素。

    建议把所有需要再次使用的引用元素定义在<defs>元素里面。这样做可以增加 SVG 内容的易读性和无障碍。 在<defs>元素中定义的图形元素不会直接呈现。你可以在你的视口的任意地方利用 use (opens new window)元素呈现这些元素。

      <svg width="300" height="300" xmlns="http://www.w3.org/2000/svg">
    <defs>
      <!-- 0.样式 -->
      <style>
        rect{
          fill: green;
        }
      </style>
      <!-- 1.定义了一个矩形 -->
      <rect id="rectangle"  x="0" y="0" width="100" height="50"></rect>

      <!-- 2.定义了一个组合图形 -->
      <g id="logo" fill="transparent" stroke="red">
        <circle cx="50" cy="50" r="25"></circle>
        <circle cx="80" cy="50" r="25"></circle>
        <circle cx="110" cy="50" r="25"></circle>
        <circle cx="140" cy="50" r="25"></circle>  
      </g>
    </defs>
  
  </svg>

    # <use>

    参考：https://developer.mozilla.org/zh-CN/docs/Web/SVG/Element/use (opens new window)

    <use>元素在 SVG 文档内取得目标节点，并在别的地方复制它们。 注意下面图中<use>元素的属性： 

      <svg width="300" height="300" xmlns="http://www.w3.org/2000/svg">
    <defs>
      <!-- 0.样式 -->
      <style>
        rect{
          fill: green;
        }
      </style>
      <!-- 1.定义了一个矩形 -->
      <rect id="rectangle"  x="0" y="0" width="100" height="50"></rect>

      <!-- 2.定义了一个组合图形 -->
      <g id="logo" fill="transparent" stroke="red">
        <circle cx="50" cy="50" r="25"></circle>
        <circle cx="80" cy="50" r="25"></circle>
        <circle cx="110" cy="50" r="25"></circle>
        <circle cx="140" cy="50" r="25"></circle>  
      </g>
    </defs>
  
  </svg>


<svg width="300" height="300" xmlns="http://www.w3.org/2000/svg" >
  <!-- 
    他的宽和高是没有生效的! 只用 use 引用的图形是 svg 或 symbol 才会起作用
   -->
  <use href="#rectangle" width="300" height="100" ></use>
</svg>


<svg width="300" height="300" xmlns="http://www.w3.org/2000/svg" >
  <use href="#logo"></use>
</svg>

    # <symbol>

    参考：https://developer.mozilla.org/zh-CN/docs/Web/SVG/Element/symbol (opens new window)

    元素<symbol>和元素<defs>类似，也是用于定义可复用元素，然后通过<use>元素来引用显示。

    <defs> 和 <symbol> 元素中定义的图形默认都不会显示在界面上。

    `<symbol>`元素拥有`viewbox`、`preserveAspectRatio`等属性。由于其拥有`viewbox`属性，所以其可以用来放大缩小（用户坐标系），所以其通常用于制作 SVG 精灵图。

    <defs>和<symbol>的区别最重要的一点就是：<defs>没有viewbox属性，而symbol拥有。

      <svg width="300" height="300" xmlns="http://www.w3.org/2000/svg" >
    
    <!-- 1.ICON previous-->
    <symbol id="previous" viewBox="0 0 100 100">
      <path fill="currentColor" d="M 80 0,L 20 50, L 80 100 Z"></path>
    </symbol>

    <!-- 2.ICON next -->
    <symbol id="next" viewBox="0 0 100 100">
      <polygon points="20 0, 80 50, 20 100"></polygon>
    </symbol>

    <!-- 复用 -->
    <!-- <use href="#previous" width="100" height="100"></use> -->
  </svg>


  <!-- 复用 -->
  <svg width="300" height="300" xmlns="http://www.w3.org/2000/svg" >
    <!-- 直接在use上指定ICON的 width和 hegiht -->
    <use href="#previous" width="50" height="50"></use>
  </svg>

  <!-- 这个属于缩小 -->
  <svg width="30" height="30" xmlns="http://www.w3.org/2000/svg" >
    <use href="#previous" ></use>
  </svg>

  <!-- 属于放大 -->
  <svg width="200" height="200" xmlns="http://www.w3.org/2000/svg" >
    <use href="#previous" ></use>
  </svg>
