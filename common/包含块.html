    参考：https://developer.mozilla.org/zh-CN/docs/Web/CSS/Containing_block (opens new window)

    css 中元素的尺寸和位置，会受它的包含块所影响。对于一些属性，例如width (opens new window)，height (opens new window)，padding (opens new window)，margin (opens new window)，绝对定位元素的偏移值（比如 position (opens new window) 被设置为 absolute 或 fixed），当我们对其赋予百分比值时，这些值的计算值，就是通过元素的包含块计算得来。

    # 确定包含块

    确定一个元素的包含块的过程完全依赖于这个元素的position属性：

    position的取值有：static（默认）、relative、absolute、fixed、sticky。

    1. 如果position属性为static、relative或sticky，包含块是由它的 **最近的祖先块元素 **的 **内容区边缘 **组成。
    2. 如果position属性为absolute,包含块就是由它 最近的**position**值 _不 _是**static****的祖先元素 **的 **内边距区边缘 **（注意是 内边距，而非 内容区）组成。
    3. 如果position属性为fixed，包含块则是页面视口[Viewport](https://developer.mozilla.org/zh-CN/docs/Glossary/Viewport)（如果是分页媒体则是该页的分页区域）。

    此外，还有下面这种特殊情况。

    1. 如果position属性为**absolute**、**fixed**，包含块也可能由满足以下条件的 **最近父元素 **的 **内边距区边缘 **组成：
      1. transform (opens new window) 或 perspective (opens new window) 的值不是 none
      2. will-change (opens new window) 的值是 transform 或 perspective
      3. filter (opens new window) 的值不是 none 或 will-change 的值是 filter（只在 Firefox 下生效）。
      4. contain (opens new window) 的值是 paint（例如：contain: paint;）
      5. backdrop-filter (opens new window) 的值不是 none（例如：backdrop-filter: blur(10px);）

    **注意：**根元素<html>所在的包含块是一个被称为 **初始包含块 **的矩形。它的尺寸是 **视口 **或 分页媒体。

    什么是连续媒体、分页媒体 在CSS中，媒体（Media）是指用于呈现文档的设备或媒介，例如屏幕、打印机、投影仪等。根据媒体类型的不同，CSS允许我们为不同的媒体类型定义不同的样式规则，以便文档在不同的设备上以最佳的方式呈现和打印。 在CSS中，媒体类型可以分为两类：连续媒体（Continuous Media）和分页媒体（Paged Media）。 连续媒体指的是一种无限延伸的媒体类型，例如屏幕、电视等。在连续媒体的情况下，文档的内容会根据视口的大小来进行布局和呈现。因此，当元素的position属性为fixed时，在连续媒体的情况下，其包含块是视口（viewport）。这意味着，元素的位置会相对于视口进行定位，而不是相对于文档流中的其它元素进行定位。 分页媒体指的是一种按页进行布局和呈现的媒体类型，例如打印机、PDF阅读器等。在分页媒体的情况下，文档的内容会被分割成一页一页的页面，并根据页面的大小和方向进行排版。因此，元素的包含块会相对于当前的分页区域（page area）进行计算。在分页媒体中，元素的position属性为fixed时，其包含块就是当前的分页区域，而不是文档流中的其它元素。 总之，连续媒体和分页媒体是CSS中的两种不同的媒体类型，它们在布局和呈现文档时有着不同的特点和规则。在连续媒体的情况下，元素的包含块是视口；在分页媒体的情况下，元素的包含块是当前的分页区域。

    # 根据包含块计算百分值

    计算方式有以下两种：

    1. 要计算 height (opens new window)top (opens new window) 及 bottom (opens new window) 中的百分值，是通过包含块的height值来计算它的百分值。
    2. 要计算 width (opens new window),left (opens new window), right (opens new window), padding (opens new window), margin (opens new window) 这些属性由包含块的 width 属性的值来计算它的百分值。
