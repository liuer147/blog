    描边动画主要是借助stroke-dasharray以及stroke-dashoffset来实现。基本步骤如下：

    1. 先将描边设置为虚线。
    2. 接着将描边偏移到不可见处。
    3. 通过动画让描边慢慢变为可见，这样就产生了动画效果。

    此外，针对 SVG 元素路径长度，可以通过元素的getTotalLength()方法获取。

    <style>
  #line1{
    /* 指定为虚线：100px实线 100px空白 */
    stroke-dasharray: 100px; 
    /* 不可见 */
		stroke-dashoffset: 100px;  /*      向起点方向移动 */
    animation: line1Move 2s linear forwards;
  }
  @keyframes line1Move {
    100%{
      stroke-dashoffset: 0px;  /* 可见 */
    }
  }
</style>

<svg width="300" height="300" xmlns="http://www.w3.org/2000/svg" >
    <!-- 
      stroke , 而不是 fill
     -->
    <line 
      id="line1"
      x1="100" y1="70"  x2="200" y2="70" 
      stroke="red" 
      stroke-width="10"
    >
    </line>
  </svg>

    解决方案：

    1. 通过对应 SVG 元素的getTotalLength()方法获取。
    2. 再分别将描边设置为虚线。并且偏移到看不见，再通过帧动画慢慢显示即可。
    
<style>
  .outline {
    stroke-dasharray: 1019;
    stroke-dashoffset: 1019;
    animation: moveStick 2s linear forwards;
  }
  @keyframes moveStick {
    100% {
      stroke-dashoffset: 0;
    }
  }
  .inside-l,
  .inside-r {
    stroke-dasharray: 486;
    stroke-dashoffset: 486;
    animation: moveStick 1.5s linear forwards;
  }
  .inside-l {
    animation-delay: 1.5s;
  }
  
  .stick {
    stroke-dasharray: 252;
    stroke-dashoffset: 252;
    animation: moveStick 1s linear forwards;
    animation-delay: 3s;
  }
  
  .drop {
    stroke-dasharray: 36;
    stroke-dashoffset: 36;
    animation: moveStick 0.5s linear forwards;
    animation-delay: 4s;
  }
</style>

<script>
  window.onload = function () {
    getPathLength("stick"); // 252px
    getPathLength("drop"); // 36
    getPathLength("outline"); // 1019
    getPathLength("inside-l"); // 486
    getPathLength("inside-r"); // 486
  };

  function getPathLength(className) {
    let stickEl = document.getElementsByClassName(className)[0];
    let stickLength = stickEl.getTotalLength();
    console.log(className + "Length=", stickLength);
  }
</script>
