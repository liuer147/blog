    # 概述

    压缩命令 应用场景
    tar linux最常用的压缩命令，用于备份。备份或压缩首选
    gzip 配合其他命令的压缩，未来数据库备份中使用
    zip/unzip 一般用于解压来自windows的zip格式的压缩包

    # tar

    选项 含义
    c create，打包，通过tar命令进行打包
    z gzip，压缩，通过gzip工具进行压缩 -- tar.gz
    v verbose，显示过程
    f file，指定压缩包，选项在最后。
    t list，查看需要
    x 解压

    # 打包压缩

    • windows：压缩
    • linux：打包（文件几种在一起），压缩（解压空间）
    • tar命令格式： tar 选项 压缩之后存放的路径 被压缩的文件或目录
    • 了解： --exclude=xxx压缩时排除xxx
    • 了解：--exclude-from=xxx压缩时，排除出现在文件xxx中的内容
    #案例1 打包压缩/etc/目录，存放在/tmp/etc.tar.gz
[root@liuer ~]# tar czvf /tmp/etc.tar.gz /etc
... # 过程
[root@liuer ~]# ll -h /tmp/etc.tar.gz
-rw-r--r-- 1 root root 5.5M Oct  5 14:36 /tmp/etc.tar.gz

    # 解压

    • 解压命令 默认解压到当前目录。
    • -C选项可以指定解压目录
    ## 可以使用 zxvf, 也可以使用 xf
[root@liuer tmp]# tar xf etc.tar.gz
[root@liuer tmp]# ls
etc  etc.tar.gz  systemd-private-3c3bff38bf944ea1ac5084b8f3a265ad-chronyd.service-nIi2dj

## 解压到指定目录
[root@liuer tmp]# tar xf etc.tar.gz -C /self_dir
[root@liuer tmp]# ll /self_dir
total 848
...

    # gzip

    • gzip常用于压缩某一个文件
    • gzip会原地压缩/解压缩，操作之后原文件没了（在压缩包内）
    • 有时候想关闭某些配置文件，就可以使用gzip压缩
    gzip xxx ## 压缩
gzip -d xxx ## 解压缩

    # zip/unzip

    # zip

    • zip压缩：zip xxx，压缩之后原文件仍然存在
    • 默认不会压缩目录，需要借助-r选项(递归压缩)才行。

    # unzip

    • unzip解压：unzip xxx，默认解压到当前文件。
    • unzip -d可以指定解压到指定目录
