    别名

    # 别名

    • alias：查看系统别名
    • alias a：查看别名a的信息
    • alias xx='命令'：创建别名 -> 临时生效，重新登录失效
    ## 创建rm别名   -> 覆盖了rm命令				-> 测试的时候不要拿/来试，拿具体文件
## 等号两边不要有空格，等号右边得是命令或者脚本
alias rm='echo 请不要使用rm命令'
    1
    2
    3

    怎么临时取消别名？

    • 命令绝对路径： /bin/rm xx.txt
    • 命令前加个转义字符： \rm xx.txt

    怎么让别名永久生效呢？

    • 仅root用户生效
      1. vi ~/.bashrc修改用户的别名文件
      2. 将原先的别名注释#，将新的别名写入
    • 所有用户都生效
      1. 查看~/.bashrc文件里面是否已经存在相同名称的别名了，如果已经存在，则注释。
      2. 在/etc/profile文件里面写入对应别名
      3. 重启生效

    # dos2unixwindows和linux文件的格式转换

    默认可能没有安装，执行yun install -y dos2unix安装

    • 本质转换的是文件的回车换行符号
    • 应用场景：
      • 在windows下编写脚本（代码），上传到linux使用与执行。执行失败了
      • dos2unix转换下这个文件，然后重新执行即可
