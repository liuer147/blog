    sudo权限，给普通用户授权以root身份执行命令。

    # 进行sudo授权

    root用户进行授权操作

    • 授权命令通常格式：用户名 ALL=(ALL) 命令
      • 授权test用户所有权限：test ALL=(ALL) ALL -- 慎重授予
      • 通常情况下，在使用sudo时，会提示输入当前用户密码，修改其不需要设置密码：test ALL=(ALL) NOPASSWD: ALL
      • 多个命令以,隔开：test ALL=(ALL) /bin/vi, /bin/rm
      • 授予排除部分命令!以外的所有命令：test ALL=(ALL) ALL, ! /bin/vi, ! /bin/rm

    授权的命令： visudo 编辑sudo命令 -> 相当于 vi /etc/soduers

    不过 visudo 在格式出错时会出现一些提示，而直接使用vi编辑，则没有

    1、进入之后，执行100gg快速定位到100行，root授权命令所在区域 2、添加命令，比如给test用户授权root身份的vi命令:test ALL=(ALL) /bin/vi

    [root@liuer ~]$ visudo # 会进入授权文件，添加指令  test 	ALL=(ALL)	/bin/vi 即可授权
[root@liuer ~]$ grep test /etc/sudoers
test 	ALL=(ALL)	/bin/vi
