    # 基本命令

    # pwd

    打印当前目录

    # tree targetDir

    以树形结构打印目录结构，需要使用**yum install -i tree**才能使用

    # file filename

    file命令会输出文件内容的简要摘要

    # less filename

    less命令允许我们前后查看文本文件。

    上翻页键Page Up或b 后翻一页
    下翻页键Page Down或空格 前翻一页
    上方向键 向后一行
    下方向键 向前一行
    G 移动到文本文件末尾
    1G或g 移动到文本文件开头
    /characters 向前搜索指定的字符串
    n 重复上一次搜索
    h 重复上一次操作
    q 退出less指令

    # 通配符

    通配符 含义
    * 匹配任意多个字符。注意，并不是重复它前面的字符
    ? 匹配任意单个字符**。**
    [chars] 匹配属于字符集合chars中的任意单个字符
    [!chars] 匹配不属于字符集合chars中的任意单个字符
    [[:class:]] 匹配属于字符类class中的任意单个字符

    常见的字符类有：

    • [:alnum:]：匹配任意 单个字母数字 字符
    • [:alpha:]：匹配任意 单个字母
    • [:digit:]：匹配任意 单个数字
    • [:lower:]：匹配任意 单个小写字母
    • [:upper:]：匹配任意 单个大写字母

    *并不能匹配以 点号 开头的文件（隐藏文件）

    # mkdir dir...

    mkdir用于创建目录

    ...表示可以重复出现

    mkdir dir1 // 创建单个目录
mkdir dir1 dir2 dir3 // 创建多个目录

    创建文件，不过本质上是 修改文件的时间戳。

    1

    # cp item1... targetDir

    cp命令可以将多个文件/目录复制到targetDir目录中

    • 复制单个文件/目录时，如果targetDir不存在，则会自动创建
    • 赋值多个文件/目录时，targetDir必须存在

    cp命令常用选项：

    选项 含义
    -a, --archive 复制文件和目录及其包括的 所有权与权限 在内的所有属性
    -i, --interactive 在覆盖文件之前，提示用户确认。
    -r, --recursive 递归 复制目录及其内容
    -u, --update 只复制 目标目录中存在的 或 相比目标目录中现有文件更新的文件 ，选项只在 复制大批量文件时很实用 ，因为他能够跳过那些不必复制的文件
    -v, --verbose 在复制时显示相关信息
    -t 有此选项之后，命令的形式变为了cp target file...

    # mv item... target

    mv命令可以执行 文件移动 和 文件重命名 操作。命令操作之后，原先的文件名不再存在。

    mv item1 item2 // 将文件或目录item1  移动或重命名 为item2

mv item... targetDir // 将多个文件或目录移动到另一个目录
    1
    2
    3

    mv命令常用选项：

    选项 含义
    -i, --interactive 在覆盖已有文件之前，提示用户确认。
    -u, --update 只移动目标目录中不存在或比目标目录中现有文件更新的文件
    -v, --verbose 移动时显示相关信息

    # rm item...

    rm命令可用于删除文件和目录。 rm命令常用选项：

    选项 含义
    -i, --interactive 在覆盖已有文件之前，提示用户确认。
    -r, --recursive 递归 删除目录
    -f, --force 忽略不存在的文件，不提示。此选项会屏蔽--interactive选项
    -v, --verbose 删除时显示相关信息

    注意： 当想删除某些后缀文件时，采用rm *.html命令。当无意中在*和html之前多加了一个空格，从而变成了rm * .html。此时，rm命令会删除目录中的所有文件。这种情况非常严重，rm 命令删除的文件是无法撤销删除的。 所以：只要使用rm命令时用到了通配符，处理仔细检查输入内容以外，还要先使用ls测试一下通配符，知道要删除那些文件后；再按向上方向键调出之前的命令，替换ls为rm。
