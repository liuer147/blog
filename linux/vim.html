    # 控制光标

    • 上下左右
    h 向左移动
    l 向右移动
    j 向下移动
    k 向上移动
    • 移动光标
    $ 移动到行首
    ^ 移动到行尾
    G 将光标移动到文件的最后一行
    gg 将光标移动到文件的第一行
    100gg 将光标移动到文件的第100行，具体可变
    • 复制粘贴删除
    yy 复制当前行
    p 粘贴
    dd 剪切当前行，可用于删除
    • 搜索查找，man/vi/vim很多地方通用
    /内容 向下搜索
    ?内容 向上搜索
    n 继续向下搜索
    N 继续向上搜索
    • 替换
    :%s#str1#str2#g 将文件中所有的str1替换成str2
    • 批量处理功能
    2dd 删除两行，其实很多快捷键前面都可以加上数字
    1、shift + v，2、d 1、选中多行内容，2、删除
    1、ctrl + v，2、d 1、选中多行开头内容，2、删除。可以使用上下左右进行选择
    • 其他快捷键
    u 撤销
    :noh 临时取消高亮显示
    :set nu 显示行号
    :set nonu 取消显示行号
    C 删除光标到行尾的内容，并进入编辑模式
    A 将光标移动到结尾，并进入编辑模式
    o、O 在当前行下面o/上面O插入一行，并进入编辑模式
